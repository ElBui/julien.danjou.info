--- title: "mod_defensible" --- {% mark excerpt %} *mod_defensible* is an Apache 2.x module intended to block spammers using DNSBL servers. {% endmark %} It will look at the client IP and check it in one or several DNSBL servers and return a 403 Forbidden page to the client. # Download You can download it from [the Git repository](http://git.naquadah.org/?p=mod_defensible.git;a=summary). # Installation 
% ./configure
% make
% su -
# make install
You can use =--with-udns= to enable [udns](http://www.corpit.ru/mjt/udns.html) (asynchronous resolver library) usage, which should be faster. However, this is not mandatory. 
# Configuration
# Activate DNSBL usage
DnsblUse On
# Specify DNSBL servers
DnsblServers xbl-bl.spamhaus.org. mydnbl.server.org.
With udns support, you can also use: 
# Specify another nameserver to use instead of default system resolver
DnsblNameserver 192.168.3.254
